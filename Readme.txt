**Título del Proyecto**
Desarrollo de un pqrs para prueba

**Comenzando**
Estas instrucciones te permitirán obtener una copia del proyecto en 
funcionamiento en tu máquina local para propósitos de desarrollo y pruebas.

**Pre-requisitos**
Instalar un servidor Local en tu equipo por ej:
XAMPP Apache + MariaDB + PHP + Perl
https://www.apachefriends.org/es/index.html
Descarga el repositorio de github

**Instalación**
Mueve el reposiorio a la carpeta donde instalaste tu servidor local:
ej:C:\xampp\htdocs\

importa la base de datos

Cambia los datos de los archivos conexion.php y conexion1.php
pqrs\config\
ej:
$host = 'localhost';
$username = 'root';
$passwd = '';
$dbname = 'pqrs';

Inicia tu servidor local
Finaliza ingrensado desde el navegador al login como llamaste la carpeta ej:
http://localhost/pqrs/index.php
